# Edna
**Edna** is a simple, yet strapping, CSS framework that uses [LESS][1].

**Edna**

 - An HTML reset: [normalize.css][2].
 - CSS styling for basic HTML elements.
 - A page layout system.
 - CSS styling based on [SMACSS][3].
