Default

We have three different breakpoints, or four different sizes for widgets and page layout. We control the sizes of these breakpoints from the variables, so change them in one spot and they all change.

// MEDIA QUERY DIMENSIONS
// unit gets applied later
@mediaquery-small: 320;
@mediaquery-medium: 599;
@mediaquery-large: 995;
@mediaquery-xlarge: 1200;

Here are our four media queries. You can see that we add the unit, as well as 1 pixel in certain spots, to the variable value. Notice that we also set a min-width for the body.

body { min-width: unit(@mediaquery-small, px); }

// SMALL
@media @mq-sm {
    // here be dragons
}

// MEDIUM
@media @mq-md {
    // here be ghouls
}

// LARGE
@media @mq-lg {
    // here be hooligans
}

// XLARGE
@media @mq-xl {
    // here be goblins
}

All of the above outputs the following.


body {
  min-width: 320px;
}
// SMALL
@media screen and (max-width: 599px) {
    // here be dragons
}

// MEDIUM
@media screen and (min-width: 600px) and (max-width: 995px) {
    // here be ghouls
}

// LARGE
@media screen and (min-width: 996px) and (max-width: 1199px) {
    // here be hooligans
}

// XLARGE
@media screen and (min-width: 1200px) {
    // here be goblins
}
