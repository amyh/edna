Media Blocks

The "media block" is a common widget layout of two or more bits of content side by side without wrapping. A common use is for an image and text combination such as a logo and heading or page title.

There are two type of media block layouts. One (class="media-block") allows for vertically centering content and uses the CSS display: table/table-row/table-cell properties. The other (class="media-float") uses floats.

Why two types? Because.

Each type of media block has different properites which are outlined below. So choose wisely when selecting a media block type.

Media Block (display table properties)

For this media block, the markup mimics a table. The class media-block represnets the <table>, the class media-row represents the <tr> and media-cell represents the <td>.

By default, media-cell content is centered vertically (just like a table cell). The following classes can be used to change the vertical alignement of the media-content: media-top, media-btm.

Media Float (float properties)

The media block using CSS floats is a bit different. The content is always vertically aligned at the top, can have slightly less markup, content can be floated left or right and there are other things I am just not thinking of at the moment.
