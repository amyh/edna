Lists

DNA Plus includes classes to allow for changing the list style types of ordered (OL) and unordered (UL) lists.

UL Style Types

The following classes can be used to define/change the list-style-type of a UL: disc, circle, square.

  • Disc List Item 1
  • Disc List Item 2
  • Disc List Item 3
  • Circle List Item 1
  • Circle List Item 2
  • Circle List Item 3
  • Square List Item 1
  • Square List Item 2
  • Square List Item 3

These classes can be used on nested lists to make them all the same, change the order or extend the pattern.

  • Square List Style
    • Disc List Style
      • Circle List Style
        • Circle List Style

OL Style Types

The following classes can be used to define/change the list-style-type of a OL: decimal, lower-alpha, upper-alpha, lower-roman, upper-roman.

  1. Decimal (default) List Item 1
  2. Decimal (default) List Item 2
  3. Decimal (default) List Item 3
  1. Lower Alpha List Item 1
  2. Lower Alpha List Item 2
  3. Lower Alpha List Item 3
  1. Upper Alpha List Item 1
  2. Upper Alpha List Item 2
  3. Upper Alpha List Item 3
  1. Lower Roman List Item 1
  2. Lower Roman List Item 2
  3. Lower Roman List Item 3
  1. Upper Roman List Item 1
  2. Upper Roman List Item 2
  3. Upper Roman List Item 3

These classes can be used on nested lists to make them all the same or change the order or extend the pattern.

  1. Upper Roman List Style
    1. Upper Alpha List Style
      1. Lower Roman List Style
        1. Lower Alpha List Style
          1. Decimal List Style

Base List

The following class can be used to completely reset a list so it has no margin and no padding.

  • Decimal (default) List Item 1
  • Decimal (default) List Item 2
  • Decimal (default) List Item 3
