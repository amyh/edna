Form Widgets

Search with Button

To create the Search and Button combo, the elements need to be wrapped within a container with a class="btn-group"

<div class="btn-group">
	<input class="grp-input" type="search" />
	<input class="grp-btn" type="submit" />
</div>

Multi Buttons

To create the multi button combo, the elements need to be wrapped within a container with a class="btn-group"

This example uses :first-child and :last-child to position the Submit and Rest buttons.

<div class="btn-group">
	<input class="grp-input" type="search" />
	<input class="grp-btn" type="submit" />
	<input class="grp-btn" type="reset" />
</div>

Note: :last-child does not work IE8 and earlier.

For IE8 and earlier we utilize .first and .last classes.

<div class="btn-group">
	<input class="grp-input first" type="search" />
	<input class="grp-btn" type="submit" />
	<input class="grp-btn last" type="reset" />
</div>

Dropdowns

In order to use styled dropdowns we are creating our own rather then use a select menu.

These use hidden radio buttons (or checkboxes if you desire a multi-select dropdown) with a selectable label.

<div class="dd">
	<button class="dd-active">
		<span class="txt">One</span>
		<div class="transport">
			<div class="bottom"></div>
		</div>
	</button>
	<ul class="dd-list">
		<li class="dd-list-item">
			<input class="dd-input" type="radio" name="list" id="one" />
			<label class="dd-label" for="one">One</label>
		</li>
	</ul>
</div>

These will seamlessly fit into our multi-button groups.

<div class="btn-group">
	<div class="dd">

		</ul>
	</div>
	<input class="grp-btn" type="submit" />
