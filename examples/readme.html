EDNA

EDNA is a simple, yet strapping, CSS framework that uses LESS. It comes in three parts:

  1. EDNA
    • An HTML reset: normalize.css.
    • CSS styling for basic HTML elements.
    • A page layout system.
  2. Plus
    • CSS styling based on SMACSS.
    • Base Widgets
      • These will make up Composite Widgets.
    • Composite Widgets
      • Complex widgets, usually made up of multiple Base Widgets.
